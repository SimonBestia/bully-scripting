# GetCutsceneRunning

## Description

Returns the numeric ID of the cutscene currently playing. Returns 0 if no cutscene is active.

```lua
function GetCutsceneRunning() --[[ ... ]] end
```

## Parameters

None.

## Return Values

- `cutscene`: _`integer`_ - The cutscene ID, or 0 if none is running.
