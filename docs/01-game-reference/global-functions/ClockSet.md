# ClockSet

## Description

Sets the current in-game time to the specified hours and minutes.

```lua
function ClockSet(hours, minutes) --[[ ... ]] end
```

## Parameters

- `hour`: _`integer`_ - Hours to set, in 24-hour format (0–23).
- `minute`: _`integer`_ - Minutes to set (0–59).

## Return Values

None.

None.
